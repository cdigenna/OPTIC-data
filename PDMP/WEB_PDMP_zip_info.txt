OPTIC-Vetted PDMP Policy Data

The files in this folder are for the Prescription Drug Monitoring Program (PDMP) policy variables (through June 1, 2017) included in the OPTIC-Vetted Policy Data Sets. 	
Analytic file for Stata:  WEB_PDMP.dta
Analytic file for Excel:  WEB_PDMP.xlsx
Documentation file: WEB_PDMP.pdf

Last Updated: July 29, 2020

The folder is associated with "The State of the Science in Opioid Policy Research" https://www.rand.org/pubs/external_publications/EP68218.html. 

All OPTIC-Vetted Policy Data Sets available for download can be found at https://www. rand.org/health-care/centers/optic/resources/datasets.html.
