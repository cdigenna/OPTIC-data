OPTIC-Vetted Naloxone Policy Data

The files in this folder are for the Naloxone Access law variables (1990-2017) included in the OPTIC-Vetted Policy Data Sets. 	
Analytic file for Stata:  WEB_NAL.dta
Analytic file for Excel:  WEB_NAL.xlsx
Documentation file: WEB_NAL.pdf

Last Updated: July 29, 2020


The folder is associated with "Legal Requirements and Recommendations to Prescribe Naloxone" https://www.rand.org/pubs/external_publications/EP68090.html.

All OPTIC-Vetted Policy Data Sets available for download can be found at https://www. rand.org/health-care/centers/optic/resources/datasets.html.
